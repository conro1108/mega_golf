# mega_golf

Pixel-art minigolf. Physics-driven putting across an 18-hole course — front
nine easier, back nine harder, both perspectives, ending in a sprawling mega
hole. See `DESIGN.md` for the why.

`npm run dev` to play. Best run per hole is saved locally and replays as a
translucent ghost; the scorecard tracks your first attempt per hole each
session. Online scorecards/ghosts are a later layer — see DESIGN.md.
